# ToyOS
Toy Operating System for concept learning

## Implemented
- Serial port driver
- VGA display driver
- ATA Drive IO Driver
- Readonly USTAR "filesystem" driver
- Paging
- GDT setting
- Interrupts
## TODO
- Keyboard IO Driver
- Process scheduling
- Memory management

## How to build
Simply execute
```
./compile.sh
```
(Need GCC Cross Compiler for success)
## How to run
```
./run.sh
```
(Need QEMU for success)
